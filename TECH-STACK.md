# Tech Stack - MuniGo Mobile

## Core
- **Framework**: React Native (via Expo)
- **Lenguaje**: TypeScript
- **Navegación**: React Navigation

## Styling
- **Lupa UI System**: Diseño premium basado en Stitch.
- **Tokens**: Definidos en src/config/theme.ts.

## Deployment
- **iOS**: Apple App Store
- **Android**: Google Play Store
- **Contenedores**: Docker (para servicios backend y APIs)
